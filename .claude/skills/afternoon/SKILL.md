# Afternoon Scan

Read today's afternoon scan and present it conversationally. The scan is generated by the afternoon scan script (synthesized, orientation-first format).

## Decision Tree

### 1. Check for today's scan

```bash
TODAY=$(date +%Y-%m-%d)
SCAN_FILE=~/your-workspace/Afternoon-Scan.md
ls -la "$SCAN_FILE" 2>/dev/null
```

Also check the archive:
```bash
ls -la ~/your-workspace/Documents/Field-Notes/$TODAY-Afternoon-Scan.md 2>/dev/null
```

### 2A. Scan exists — read and present

Read the scan file with the Read tool. Present it with a one-liner opener:

- "Afternoon check. Here's what I see."
- "Mid-day sweep. Let's see where things stand."

Render the full markdown as-is. Don't summarize or truncate.

### 2B. No scan for today — report and offer

If no scan exists for today's date:

1. Tell the user: "No afternoon scan generated yet today."
2. Check if the scheduled job ran (check log files)
3. Offer to generate one manually:
   ```bash
   cd ~/your-workspace && bash .claude/scripts/afternoon-scan.sh --preview
   ```
4. If generation succeeds, read and present the output.
